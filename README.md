#Usage: "sytcs"
##Options:
###  -h    Show this help message
###  -p    password of current user 
###  -c    Signature certificate
###  -k    Specify configuration file (default: ~/Library/Keychains/login.keychain-db)
##Examples:
### "sytcs" -p '12345' -c 'Developer ID Application: Zhang San (6TC35296BK)'  -k '~/Library/Keychains/login.keychain-db' xxx.app
